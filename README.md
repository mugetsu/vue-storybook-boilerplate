# Vue Storybook + Storyshots Boilerplate

Boilerplate for Vue component library powered by Storybook + Storyshots

## Run it locally

Install packages using [yarn](https://yarnpkg.com/en/docs/install#mac-stable)

```bash
yarn
```

Run Storybook

```bash
yarn storybook
```
